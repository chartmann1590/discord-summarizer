Configuration Help

  1. Open Discord in your web browser
  2. Press F12 to open Developer Tools
  3. Go to the "Application" or "Storage" tab
  4. Find "Local Storage" → "https://discord.com"
  5. Look for the "token" entry and copy its value
Keep your token secret! It provides full access to your Discord account.

Enter one channel per line. You can optionally specify a server name for grouping:

  • Basic format: channel_id
  • With server name: channel_id,Server Name

To get channel IDs:

  1. Enable Developer Mode in Discord Settings → Advanced
  2. Right-click on any text channel
  3. Select "Copy Channel ID"
